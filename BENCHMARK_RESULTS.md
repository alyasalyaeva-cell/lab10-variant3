# Результаты тестирования производительности (Вариант 3)

В данном отчете представлено сравнение скорости работы сервисов на Go (Gin) и Python (FastAPI) на основе реальных замеров утилитой Apache Benchmark.

## Параметры тестирования
- **Инструмент:** Apache Benchmark (ab)
- **Количество запросов (-n):** 1000
- **Уровень параллелизма (-c):** 20
- **Тестовый эндпоинт:** `/ping`

## Сводная таблица результатов

| Метрика | Go (Gin) | Python (FastAPI) |
| :--- | :--- | :--- |
| **Requests per second (RPS)** | 1810.61 #/sec | 834.91 #/sec |
| **Time per request (mean)** | 11.046 ms | 23.955 ms |
| **Total time for tests** | 0.552 seconds | 1.198 seconds |
| **99% Percentile** | 33 ms | 51 ms |
| **Longest request** | 50 ms | 53 ms |

## Анализ результатов
На основе полученных данных видно, что:
1. **Производительность:** Go (Gin) обрабатывает в **2.17 раза больше** запросов в секунду (RPS), чем Python (FastAPI).
2. **Задержки:** Среднее время ответа у Go почти в **2.2 раза ниже** (11 мс против 24 мс).
3. **Стабильность:** Go показал лучший результат на 99-м перцентиле (33 мс против 51 мс), что говорит о меньшем разбросе времени выполнения при нагрузке.

**Вывод:** Тестирование подтверждает более высокую эффективность Go-сервиса в условиях многопоточной нагрузки. Однако оба фреймворка успешно справились с задачей без единого сбоя (Failed requests: 0).
