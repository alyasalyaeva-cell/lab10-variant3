## СРЕДНИЙ УРОВЕНЬ

### Задание 3: Валидация входных данных в Go
#### Промпт 1
**Инструмент:** Gemini
**Промпт:** "Как в Go (Gin) реализовать валидацию входящего JSON через структуры? Мне нужно, чтобы поле Username было не короче 3 символов, Age было не меньше 18, а Email проверялся на корректность формата."
**Результат:** Создана структура `UserProfile` с тегами `binding:"required,min=3,gte=18,email"`. Настроена обработка ошибки 400 Bad Request при нарушении правил валидации.

### Задание 5: Передача сложных JSON-структур
#### Промпт 1
**Инструмент:** Gemini
**Промпт:** "Напиши код на FastAPI (Python), который принимает произвольный JSON-объект и пересылает его (relay) в другой микросервис на Go через библиотеку httpx. Обеспечь проброс ответа обратно пользователю."
**Результат:** Реализован эндпоинт `/relay`, который успешно передает вложенные структуры данных между Python и Go сервисами.

### Задание 7: Graceful Shutdown (Безопасное завершение)
#### Промпт 1
**Инструмент:** Gemini
**Промпт:** "Реализуй Graceful Shutdown для Go (Gin) с использованием каналов и сигналов ОС (SIGINT, SIGTERM). Также напиши аналогичное решение для FastAPI через lifespan."
**Результат:** - В Go добавлена блокировка через канал `os.Signal` и метод `srv.Shutdown()` с таймаутом.
- В Python использован `asynccontextmanager` для корректного закрытия ресурсов при остановке контейнера/процесса.
