## Тема: Сравнение FastAPI (Python) и Gin (Go): Валидация и Graceful Shutdown

### Описание
Проект представляет собой систему из двух взаимодействующих микросервисов. В ходе работы реализованы механизмы строгой валидации данных на стороне Go и обеспечено безопасное завершение работы обоих приложений (Graceful Shutdown).

### Реализованные задачи (Средняя сложность):
1. **Задание 3:** Валидация JSON в Go (проверка email, возраста и длины имени).
2. **Задание 5:** Передача сложных структур данных между Python и Go.
3. **Задание 7:** Graceful Shutdown для обоих сервисов (обработка SIGINT/SIGTERM).
4. **Задание 6:** Сравнение производительности через Apache Benchmark.
