### Описание проекта
Проект состоит из двух сервисов, взаимодействующих по протоколу HTTP. 
- **Go-service (Gin):** Принимает данные, выполняет строгую валидацию и обеспечивает безопасную остановку.
- **Python-service (FastAPI):** Служит прокси-слоем для передачи сложных JSON-структур в Go.

### Выполненные задачи (Средняя сложность)
1. **Задание 3 (Валидация):** В Go реализована проверка входящего JSON (Username >= 3 символа, Age >= 18, корректный Email).
2. **Задание 5 (JSON):** Реализована передача вложенных структур данных между языками.
3. **Задание 7 (Graceful Shutdown):** - В Go используется `context` и `os.Signal` для завершения активных соединений перед выходом.
   - В Python используется `lifespan` контекст-менеджер.
4. **Задание 6 (Бенчмарк):** Проведено сравнение производительности.

1. **Запуск Go:** `cd go-service && go run main.go`
2. **Запуск Python:** `cd python-service && venv\Scripts\activate && uvicorn main:app --port 8000`
3. **Тест взаимодействия:** Отправьте POST запрос на `http://localhost:8000/relay` с телом:
